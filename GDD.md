## 1. 角色设计

### 1.1 玩家角色（骑士）

| 属性 | 当前值 | 可调？ | 说明 |
|------|--------|-------|------|
| 生命值 (HP) | 50 | ✅ | 50格血，受1点伤害 |
| 移动速度 | 200 px/s | ✅ | |
| 跳跃力度 | -450 | ✅ | 负值，值越大跳越高 |
| 攻击伤害 | 3 | ✅ | 3刀砍死9血弓箭手 |
| 攻击判定范围 | 150 px | ✅ | 近战范围（约2-3格） |
| 闪避速度 | 480 px/s | ✅ | |
| 闪避持续时间 | 0.25 s | ✅ | |
| 闪避无敌帧 | 1.2 s | ✅ | |
| 攻击冷却（连击计时） | 0.6 s | ✅ | combo_timer |
| 受伤闪烁颜色 | 红色 (3,1,1) | ✅ | |

**操作按键：**
| 按键 | 动作 |
|------|------|
| A / D | 左右移动 |
| Space | 跳跃 |
| 鼠标左键 | 攻击 |
| X | 闪避（无敌帧） |

**动画状态：**
- idle / run / jump / crouch-idle / attack1 / attack2 / attack3 / rip（死亡）

## 6. 协作说明

### 6.1 Git 协作流程
```bash
git clone https://github.com/wyzhou01/godot-platformer.git
# 用 Godot 打开 project.godot
# 改完后
git add .
git commit -m "描述改动"
git push
```

### 6.2 注意事项
- `.godot/` 和 `*.import` 已被 `.gitignore` 忽略，**不要**提交
- 场景文件 `.tscn` 和脚本 `.gd` 是纯文本，可以 diff/merge
- 代码风格：GDScript，4空格缩进
