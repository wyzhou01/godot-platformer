## 1. 角色设计

### 1.1 玩家角色（骑士）

| 属性 | 当前值 | 模式 | 说明 |
|------|--------|------|------|
| 生命值 (HP) | 1 | 即死模式 | **被命中立即死亡**（简化设计） |
| 移动速度 | 200 px/s | ✅ | |
| 跳跃力度 | -450 | ✅ | 负值，绝对值越大跳越高 |
| 攻击伤害 | 3 | ✅ | 3刀砍死 9HP 弓箭手 |
| 攻击判定半径 | 80 px | ✅ | 近战范围 |
| 闪避速度 | 480 px/s | ✅ | |
| 闪避持续时间 | 0.25 s | ✅ | |
| 闪避无敌帧 | 1.2 s | ✅ | |
| 攻击判定 | 一次性/挥 | ✅ | 同一刀不会命中多次（`_has_hit_this_swing` 锁） |
| 受伤闪烁 | modulate Color | ✅ | white → (2.5,2.5,2.5) → white |

**操作按键**:
| 按键 | 动作 |
|------|------|
| A / D | 左右移动 |
| Space | 跳跃 |
| 鼠标左键 | 攻击 |
| X | 闪避 |

**动画状态**:
- `idle` / `run` / `jump` / `attack` / `dodge` / `die`

> 📝 计划改造：用 XSM 状态机替代 bool 标志位（`is_attacking / is_dodging / is_invincible / is_dead`）

## 6. 协作说明

### 6.1 Git 协作流程
```bash
git clone https://github.com/wyzhou01/godot-platformer.git
cd godot-platformer
# 用 Godot 4.6 打开 project.godot
# 改完后
git add .
git commit -m "描述改动"
git push
```

### 6.2 注意事项
- ✅ `.godot/`、`.DS_Store` 已被 `.gitignore` 自动忽略
- ✅ `*.png.import` **应** commit（资源索引，避免 Godot 重新生成）
- ✅ 命名规范: ASCII 英文 + 下划线（不再使用中文文件名）
- ✅ 场景文件 `.tscn` 和脚本 `.gd` 是纯文本，可 diff/merge
- ✅ 代码风格: GDScript，4空格缩进

### 6.3 分支策略（2026-07-18 起）
- `main`: 永远可跑
- `refactor/stage-N`: 重构分支，做完 merge 到 main 后删除
